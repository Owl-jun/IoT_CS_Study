### 미니프로젝트 : GUI 활용한 주소록 만들기

- 프로젝트 최종목표
    - 1차 목표
        - GUI를통한 연락처 추가,조회,수정,삭제 기능
        - 정렬 기능
        - 검색 필터 기능
        - 즐겨찾기 기능
        - 중복검사 및 제거
    - 2차 목표
        - 클라우드 연동
        - 검색 자동완성 기능
        - 백업 및 복원 기능
        - 로그 기능
        - 서버-클라이언트 구조 맛보기
